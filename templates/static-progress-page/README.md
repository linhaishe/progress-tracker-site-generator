# Static Progress Page Template

This directory will contain the generated static progress tracker page.

The generated HTML must be self-contained, read-only, usable without build tooling, and openable directly in a browser.

Styling should come from the host project's `context/design.md` when available. The generator should translate design tokens, typography, spacing, density, layout, and component conventions into static CSS. If `context/design.md` is missing or underspecified, the template should fall back to a conservative readable default style.
